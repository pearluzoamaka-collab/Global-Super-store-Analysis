# Global-Super-store-Analysis
Sales analysis project using Excel, Power Query, and Power Pivot

Project Overview
This project analyzes the Jadons Superstore dataset to uncover insights into sales performance, profitability, customer behavior, and operational efficiency using Excel, Power Query, and Power Pivot.

Objectives
Perform descriptive, diagnostic, predictive, and prescriptive analytics
Identify sales and profit drivers
Analyze loss-making product sub-categories
Evaluate the impact of discounts and shipping costs
Examine sales trends over time

Tools Used
Microsoft Excel
Power Query
Power Pivot
Pivot Tables & Charts
GitHub

Data Preparation
Cleaned raw dataset using Power Query
Removed duplicates and null values
Standardized data types
Created calendar table
Built data model with relationships

Key Measures (DAX)
Total Sales
Total Profit
Total Orders
Profit Margin
Average Discount
Average Shipping Cost

Dashboard Overview
The dashboard presents:
KPI cards for overall performance
Sales distribution by category and sub-category
Diagnostic analysis of discount and shipping impact
Sales trend over time for predictive insight

Key Insights
Technology category generates the highest sales
Some sub-categories record losses despite high sales due to high discounts and shipping costs
Sales show an overall upward trend, suggesting potential future growth
Profitability varies significantly across segments and shipping modes

Recommendations
Review discount strategies for low-margin sub-categories
Optimize shipping modes to reduce operational costs
Focus on high-margin products and customer segments
Use trend analysis for demand planning
